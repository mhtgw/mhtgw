Slackware64 14.2 packages with additional packages used for development of mhtgw build are available at 
https://www.mhtgw.com/files/Slackware.
Note: please note that mhtgw build is only using slackware64-14.2 version at this point.

this directory contains 2 directories
- slackware64-14.2 
	contains the base slackware64-14.2 packages with additional packages in patches/packages/ folder
- packages
	contains the additional packages built for slackware64-14.2
